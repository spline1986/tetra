• 4000000000000000000L - длинное целое
  • 0xDEADBEEF - шестнадцатеричное число
  • 0755 - восьмеричное число
  • 0b10101010 - двоичное число (начиная с Java 7)
  • 1_000_000 - целое число; подчёркивание для удобства программиста (начиная с Java 7)
  • 3.14F - float
  • 3.14 или 3.14D - double
  • 'A' - char
  • \u0000 - \uFFFF - кодовые елиницы Юникода
